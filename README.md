# 热点浏览器使用指南

## 项目简介

热点浏览器是一个基于 FastAPI 开发的简单热点新闻浏览工具，目前支持抖音平台的热点新闻数据。

### 主要功能
- 实时获取抖音热点新闻
- 每30秒自动刷新数据
- 响应式设计，适配不同设备
- 支持点击新闻链接跳转到原页面
- 显示新闻抓取时间

## 安装步骤

### 1. 克隆项目

```bash
git clone https://github.com/yigeiwo/douyinredian.git
cd hot_news
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 启动服务

```bash
python single_file_app.py
```

服务将在 `http://localhost:5001` 上运行。

## 使用说明

### 访问前端页面

打开浏览器，访问 `http://localhost:5001`，即可看到热点浏览器的主页面。

### 平台选择

在页面顶部的平台选择器中，点击不同的平台按钮（目前只有抖音平台）来查看对应平台的热点新闻。

### 查看新闻

新闻列表会显示在页面中央，每条新闻包含：
- 新闻排名
- 新闻标题（点击可跳转到原页面）
- 新闻来源
- 抓取时间

### 自动刷新

系统会每30秒自动刷新一次新闻数据，确保您看到的是最新的热点信息。
